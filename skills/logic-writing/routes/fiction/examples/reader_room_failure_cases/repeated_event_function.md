# Failure Type
repeated_event_function

# Bad Pattern
Every chapter in a travel section uses the same event function: the group reaches a new place, receives one warning, debates whether to continue, and continues unchanged.

# Why It Fails
The event wrapper changes, but the story state and reader expectation barely move. The repetition lacks escalation, contrast, inversion, cost, or rhythm.

# Better Direction
Turn one chapter into a wrong warning, one into a cost of ignoring advice, one into a split in the group, or one into a quiet consequence instead of another equivalent debate.

# Route
Return to story contribution, scene contract, and variation pressure review.
