# Failure Type
location_changes_without_pressure_change

# Bad Pattern
A scene moves from a classroom to a market to a rooftop, but each location works the same way: two characters stand still and explain the same conflict without new access, danger, secrecy, audience, cost, or sensory constraint.

# Why It Fails
The setting names change, but the scene pressure does not. Variation is decorative rather than functional.

# Better Direction
Make the location alter what can be said, hidden, seen, risked, touched, or lost. If it does not, merge or simplify the location movement.
